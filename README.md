# ultah-ayang

Kartu ucapan ulang tahun interaktif.

Cara pakai:

1. Edit `index.html` di bagian PERSONALISASI (nilai default) jika ingin mengganti nama, pesan, foto, musik, atau PIN.
2. Jika ingin menyertakan file foto/MP3 lokal, tempatkan di folder `assets/` dan gunakan path relatif, mis: `assets/foto.jpg` atau `assets/song.mp3`.
3. Untuk deploy via GitHub Pages:
   - Push repo ke GitHub (branch `main`).
   - Buka Settings → Pages → Source: `main` / `/ (root)` → Save.
   - Akses di: `https://hkie2017.github.io/ultah-ayang` (beberapa menit setelah deploy).

Catatan:
- Browser mungkin memblokir autoplay audio. Sediakan tombol play atau interaksi pengguna.
- Jika mau saya bantu lagi (ganti teks default, menambah assets), beri tahu.
